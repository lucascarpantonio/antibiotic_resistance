# Predicting Multi-Drug Resistance: Antibiotic Susceptibility Analysis

This project explores a Kaggle dataset on antibiotic susceptibility to identify resistance patterns and predictive insights.

## Structure
- data/: datasets
- notebooks/: Jupyter notebooks
- app/: Streamlit dashboard
- reports/: outputs

## Quick start
python -m venv venv
source venv/bin/activate   # macOS/Linux
pip install -r requirements.txt
cd app
streamlit run streamlit_app.py
